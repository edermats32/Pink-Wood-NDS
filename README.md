## Installation
You figure it out...  
What? You didn't?
1. https://github.com/flashcarts/AOS/#setup
2. Download this repo as a zip.
3. Extract the zip to your sd-card, override the files.
4. Boot `NDS - WoodR4`
5. Go into settings, change theme to `NDS OS Pink`
6. Success?
